# OPTC WIKI — Base de datos de One Piece Treasure Cruise

Última actualización: 2026-01-22 15:44:18 UTC

Descripción

OPTC WIKI es un proyecto de comunidad que ofrece una base de datos completa y gratuita para el juego móvil One Piece Treasure Cruise (OPTC). Incluye información de unidades (stats), habilidades, sockets, tablas de drops, y una calculadora de daño para ayudar a planificar equipos y estimar el rendimiento en combate.

Qué incluye

- Tabla de personajes con descripciones y estadísticas.
- Calculadora de daño que permite simular el output de un equipo.
- Tablas de drops y herramientas relacionadas (por ejemplo, soporte y bases de datos externas enlazadas).
- Código frontend estático escrito en HTML y JavaScript utilizando AngularJS, Bootstrap y jQuery.

Tecnologías

El proyecto está construido como una web estática y puede alojarse en GitHub Pages. Las imágenes y algunos recursos están enlazados desde el sitio oficial del juego, por lo que es necesaria conexión a Internet para verlas.

Instalación y uso

1. Clonar o descargar el repositorio.
2. Abrir el proyecto con un servidor local (Live Preview en editores, `xampp`, `python -m http.server`, etc.) o desplegar en GitHub Pages.
3. Acceder al índice y navegar a las secciones: Damage (calculadora), Characters (tabla), Drops (drops y tablas).
